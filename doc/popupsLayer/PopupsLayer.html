Class: PopupsLayer

PopupsLayer(layer, title, {Object}, afteropt)

弹出层对象

Constructor

new PopupsLayer(layer, title, {Object}, afteropt)

配置并生成弹出层父元素.
Parameters:
Name Type Attributes Default Description
layer Object 弹出层的外层盒子配置信息，目前有以下参数
Properties
Name Type Attributes Default Description
position String.<('LEFT'|'RIGHT'|'TOP'|'BOTTOM'|'CENTER')> <optional>
 'CENTER' 指定弹出层的位置.
style Object 样式配置信息，你可以在这个对象中传递自定义的样式，命名规范遵从HTML DOM Style对象，有以下默认配置参数
title.style.background String <optional>
 #ffffff 背景颜色.
layer.style.borderRadius String <optional>
 5px 指定圆角大小.
layer.style.maxWidth String <optional>
 none 指定盒子最大宽度.
title Object 对标题栏的配置信息，目前有以下参数
Properties
Name Type Attributes Default Description
enable Boolean <optional>
 true 指定是否需要标题栏.
draggable Boolean <optional>
 true 指定标题栏是否可以拖拽移动(内容窗口也将跟随移动).
text String 标题栏文字.
template String 你可能需要更灵活的方式控制标题栏，可以自己传入一个HTML模板加入标题栏.
btnGroup Object 指定标题栏右侧的按钮组，以下参数仅为样式中提供预设的一些按钮，你可以自己在样式写入自己喜欢的 按钮样式，并将样式名加入配置对象，仅支持class类名，不支持id.
Properties
Name Type Attributes Default Description
enable Boolean <optional>
 true 是否开启右侧按钮.
close Boolean <optional>
 true 关闭按钮.
style Object 样式配置信息，你可以在这个对象中传递自定义的样式，命名规范遵从HTML DOM Style对象，有以下默认配置参数
Properties
Name Type Attributes Default Description
color String <optional>
 #666666 标题栏字体颜色.
fontSize String <optional>
 14px 标题栏字体大小.
borderBottom String <optional>
 '.4px solid rgba(75, 97, 124, 0.53)' 标题的下边框样式，取值同border-bottom
{Object} String content - 对内容区域的配置信息，目前有以下参数
content.text String 指定要在内容区域显示的文本.
content.template String 你可能需要更灵活的方式控制内容，自己传入一个HTML模板加入弹出层，它会在文本下方展示.
content.icon Object 传入一个图标显示在内容区域，目前有以下参数
Properties
Name Type Attributes Default Description
src String 图标地址.
position String.<('LEFT'|'RIGHT')> <optional>
 'LEFT' 指定图标在弹出层的文本左边还是右边.
content.btnGroup Object 指定按钮
Properties
Name Type Description
* Object 自定义命名，此命名将成为按钮id属性名.
*.text Object 按钮内容文本.
*.fn function 指定按钮点击事件函数.
*.theme String.<(ok|cancel)> 使用样式表预设的按钮主题，此参数为主题类名，你也可以依据id属性名事先 在样式表预设主题，但是请注意，这个参数只支持class类名.
content.style Object 样式配置信息，你可以在这个对象中传递自定义的样式，命名规范遵从HTML DOM Style对象，有以下默认配置参数
Properties
Name Type Attributes Default Description
color String <optional>
 #666666 内容区域字体颜色.
fontSize String <optional>
 14px 内容区域字体大小
after function <optional>
 null 指定生成弹出层后需要进行的操作.
